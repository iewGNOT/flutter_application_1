# LifeGacha

LifeGacha is an offline-first Flutter productivity app with task, Pomodoro, reward-card, gacha, character-growth, stats, streak, and achievement systems.

Phase 1 contains only the technical blueprint and scaffold. Full Drift repositories, encrypted database opening, feature workflows, and production UI are intentionally deferred to the next phase.

See `docs/lifegacha_phase1_blueprint.md` for the architecture, domain model, persistence, state management, sequence flow, and scaffold manifest.
